HTML <HTML> Tag

Definition and Usage

The <HTML> tag represents the root of an HTML document.

It contains all the other HTML elements (except <!DOCTYPE>).

This tag is compulsory for HTML type files.

Attribute(s)

Attribute Value Description
xmlns http://www.w3.org/1999/xhtml Specifies the XML namespace attribute. (If you need your content to conform to XHTML)

The <HTML> tag supports Global Attributes in HTML.

For best results, include the lang attribute for assisting search engines and browsers.

Examples and Syntax

<!DOCTYPE html>
<html>
    <head>
        <title>web title</title>
    </head>
    <body>
        <h1>this is a heading</h1>
    </body>
</html>

Default CSS Values and Settings

Most browsers display the <HTML> element with the default values below:

html {
    display: block;
}
html:focus {
    outline: none;
}
