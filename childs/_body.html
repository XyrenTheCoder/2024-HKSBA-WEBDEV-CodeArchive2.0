HTML <BODY> Tag

Definition and Usage

The <BODY> tag defines the document's body.

It contains all the contents of an HTML document such as headings, paragraphs, images, hyperlinks, table, lists etc.

There can only be one <BODY> element in an HTML document.

Attribute(s)

The <BODY> tag supports Global Attributes in HTML and Event Attributes in HTML.

Examples and Syntax

<!DOCTYPE html>
<html>
    <head>
        <title>web title</title>
    </head>
    <body bgcolor='black' color='white'>
        <h1>this is a heading</h1>
    </body>
</html>

Default CSS Values and Settings

Most browsers display the <BODY> element with the default values below:

body {
    display: block;
    margin: 8px;
}
body:focus {
    outline: none;
}
