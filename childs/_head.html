HTML <HEAD> Tag

Definition and Usage

The <HEAD> tag is a container for metadata (data about the HTML document which will not be displayed) and is placed before the <BODY> tag.

It typically defines the document title, character set, styles, scripts, and other meta information.

The tags below can be placed inside the <HEAD> element:

  • <base>
  • <link>
  • <meta>
  • <noscript>
  • <script>
  • <style>
  • <title>

Attribute(s)

The <HEAD> tag supports Global Attributes in HTML.

Examples and Syntax

<!DOCTYPE html>
<html>
    <head>
        <title>web title</title>
        <meta charset='UTF-8'>
        <base href='https://www.w3schools.com' target='_blank'>
        <link rel='stylesheet' type='text/css' href='styles.css'>
        <style>
            h1 { color: red; }
            p { color: blue; }
        </style>
        <script>
            function func() {
                return false;
            }
            func();
        </script>
        <noscript>
            <h1>Your browser does not support javascript.</h1>
        </noscript>
    </head>
    <body>
        <h1>A stylized heading</h1>
        <p>A stylized paragraph</p>
        <a href='tags/tag_base.asp'>HTML base tag</a>
        <h2>I am being stylized by style.css</h2>
    </body>
</html>

Default CSS Values and Settings

Most browsers display the <HEAD> element with the default values below:

head {
    display: none;
}
