HTML <FOOTER> Tag

Definition and Usage

The <FOOTER> tag defines a footer of a document or a section.

A <FOOTER> usually contains authorship information, contact, copyright information, sitemap and back to top links.

This tag can be used multiple times in a document.

Attribute(s)

The <FOOTER> tag supports Global Attributes in HTML and Event Attributes in HTML.

Examples and Syntax

<!DOCTYPE html>
<html>
    <head>
        <title>web title</title>
    </head>
    <body>
        <h1>content</h1>
        <footer>
            <p>copyright (c) Archie & Archisha, 2023</p>
            <p><a href='https://example.com'>contact us</p>
        </footer>
    </body>
    </html>

Default CSS Values and Settings

Most browsers display the <FOOTER> element with the default values below:

footer {
    display: block;
}
