HTML <HTML> xmlns Attribute

Definition and Usage

The xmlns attribite specifies the XML namespace for a document.

This attribute is required in XHTML, invalid in HTML4.01, and optional in HTML5.

Validator at http://w3.org does not complain when the attribute is missing in an XHTML document, The namespace "xmlns=http://www.w3.org/1999/xhtml" is default and will be added to the <HTML> if you do not include it.

Attribute Value(s)

Value Description
http://www.w3.org/1999/xhtml The namespace to use. (for XHTML documents)

Examples and Syntax

A simple XHTML document with minimum required tags:

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns='http://www.w3.org/1999/xhtml'>
    <head>
        <title>web title</title>
    </head>
    <body>
        Contents
    </body>
</html>

Syntax:

<html xmlns='http://www.w3.org/1999/xhtml'>
