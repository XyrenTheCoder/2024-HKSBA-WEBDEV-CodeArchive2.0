HTML <STYLE> Tag

Definition and Usage

The <STYLE> tag is used to define style information (CSS) for a document.

Inside the <STYLE> element specifies how HTML elements should render in a browser.

If some properties have been defined for the same selector (element) in different style sheets, the value from the last read style sheet will be used.

To link to an external style sheet, use the <LINK> tag.

Attribute(s)

Attribute Value Description
media media_query Specifies what media/device the media resource is optimized for.
type text/css Specifies the media type of the <STYLE> tag

The <STYLE> tag supports Global Attributes in HTML.

The <STYLE> tag supports Event Attributes in HTML.

Examples and Syntax

<!DOCTYPE html>
<html>
    <head>
        <style>
            body {
                font-family: Arial, Helvetica, sans-serif;
                font-size: 12px;
                line-height: 1.5em;
            }
            h1 {color: red;}
            p {color: cyan;}
        </style>
    </head>
    <body>
        <h1>contents</h1>
    </body>
</html>

Default CSS Values and Settings

Most browsers display the <STYLE> element with the default values below:

style {
    display: none;
}
