Introduction

HTML stands for HyperText Markup Language and it is a markup language used for structuring and presenting content using tags within the document on the World Wide Web. It is being used to design web pages. HTML is an abbreviation of Hypertext and Markup language. HTML5 is the fifth and current version of HTML. It has improved the markup available for documents and has introduced application programming interfaces (API) and Document Object Model (DOM).

Source: Geeksforgeeks, Wikipedia

Tags

Tags are being used to define structures and contents and they are mostly case insensitive. Below are some of the basic tags.

Structural tags

Tag Description
<!--comment--> Defines a comment.
<!DOCTYPE> Defines the document's body.
<html> Defines the root of an HTML document.
<head> Defines the head portion of the document that contains information about the document such as title.
<meta> Provides structured metadata about the document content.
<script> Places script in the document for client-side processing.
<style> Inserts style information (commonly CSS) into the head of a document.
<body> Defines the document's body.
<footer> Represents the footer of a document or a section.
